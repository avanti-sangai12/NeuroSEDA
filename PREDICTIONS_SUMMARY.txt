  What: Scan webpage for all clickable/interactive elements

  What: Sort elements by score, take top 3
  How: Descending order (highest score first)
  Why: Show the 3 most likely next actions

ELEMENT TYPE BONUSES (Why Each Gets Points)

  Button/Submit Button:     +0.25  â† HIGHEST!
  Why: Users most commonly click buttons next (after forms)
  Examples: Send, Login, Add to Cart, Submit

  Text/Email/Password:      +0.15
  Why: Form fields come in natural progression
  Examples: Email input, Phone field, Password field

  Textarea:                 +0.12
  Why: Users fill textareas after starting forms
  Examples: Message box, Compose area, Comment field

  Link (<a> tag):           +0.10
  Why: Links are common navigation targets
  Examples: "Continue", "Next", "Learn More"

  Others (checkboxes, etc): +0.00
  Why: Less common as immediate next action


SIZE BONUS

  Width > 50px AND Height > 20px:  +0.05
  Why: Larger visible elements more likely to interact with
  Effect: Filters out decorative/tiny elements


RANDOMNESS

  Random value: -0.05 to +0.10
  Why: Makes predictions realistic and varied
  Effect: Same page gives slightly different predictions each time


FINAL CLAMP

  Result must be between: 0.1 (minimum) and 0.95 (maximum)
  Why: Ensures consistency and prevents extreme values
